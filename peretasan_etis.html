Halaman ini berisi praktikum Ethical Hacking dan Intrusion Detection System (IDS) yang dilakukan pada lingkungan lab tertutup.

⚠️ Semua aktivitas hanya untuk edukasi dan pembelajaran.

Snort adalah Network Intrusion Detection System (NIDS) yang bekerja dengan menganalisis paket jaringan berdasarkan rule.

Langkah 1 — Update Sistem

sudo apt update && sudo apt upgrade -y

Langkah 2 — Instalasi Snort

sudo apt install snort -y

Langkah 3 — Verifikasi Instalasi

snort -V

Langkah 4 — Tentukan Interface

ip a

Langkah 5 — Edit snort.conf

sudo nano /etc/snort/snort.conf

Set variabel HOME_NET sesuai IP lokal.

Langkah 6 — Buat Rule Lokal

sudo nano /etc/snort/rules/local.rules

Langkah 7 — Tambahkan Rule Uji

alert icmp any any -> $HOME_NET any (msg:"ICMP Ping Detected"; sid:1000001; rev:1;)

Langkah 8 — Test Konfigurasi

sudo snort -T -c /etc/snort/snort.conf

Langkah 9 — Jalankan Snort

sudo snort -A console -q -c /etc/snort/snort.conf -i eth0

Langkah 10 — Simulasi Serangan

ping <IP_TARGET>

Langkah 11 — Analisis Alert

Jika rule berjalan dengan benar, Snort akan menampilkan alert ICMP Ping Detected pada terminal.
