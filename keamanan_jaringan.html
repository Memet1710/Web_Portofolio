Halaman ini merangkum praktikum homelab: Honeypot, Wazuh (Manager & Agent), Suricata IDS, dan Kali Attacker. Gunakan hanya pada lingkungan uji/izin resmi.

Kali (Attacker)
Ubuntu (Honeypot)
Ubuntu (Suricata IDS)
Ubuntu (Wazuh Manager)
Ubuntu/Win (Wazuh Agent)

Tujuan: Menetapkan alamat IP statis agar setiap mesin memiliki alamat tetap dan koneksi antar layanan keamanan berjalan stabil. 

ip a
sudo nano /etc/netplan/01-netcfg.yaml
network:
  version: 2
  renderer: networkd
  ethernets:
    eth0:
      dhcp4: no
      addresses:
        - 192.168.1.10/24
      gateway4: 192.168.1.1
      nameservers:
        addresses: [8.8.8.8, 8.8.4.4]
sudo netplan apply
ip a
sudo apt update
sudo apt install git python3-venv -y
git clone https://github.com/cowrie/cowrie.git
cd cowrie
python3 -m venv cowrie-env
source cowrie-env/bin/activate
pip install -r requirements.txt
bin/cowrie start
curl -sO https://packages.wazuh.com/4.x/wazuh-install.sh
sudo bash ./wazuh-install.sh -a
curl -sO https://packages.wazuh.com/4.x/wazuh-agent.deb
sudo WAZUH_MANAGER="192.168.1.20" dpkg -i wazuh-agent.deb
sudo systemctl enable --now wazuh-agent
